## ✨ Deskripsi Singkat

Aplikasi web ini dikembangkan menggunakan **R-Shiny** sebagai tugas akhir 
mata kuliah Komputasi Statistik.

Tujuan utama aplikasi adalah memudahkan pengguna baik mahasiswa maupun peneliti dalam melakukan **analisis RAK**.

Analisis mencakup:
- Validasi data dan pengecekan desain RAK
- Visualisasi boxplot dan Q-Q plot
- Uji ANOVA dua arah (perlakuan & kelompok)
- Uji lanjut BNT, BNJ, dan Duncan
  
---

## 🚀 Fitur Aplikasi

- ✅ Upload file dataset (.csv atau .xlsx)
- ✅ Pemilihan kolom perlakuan, kelompok, dan respon
- ✅ Validasi desain RAK (setiap perlakuan muncul tepat satu kali per kelompok)
- ✅ Uji normalitas galat (Shapiro-Wilk pada residual model RAK)
- ✅ Uji homogenitas ragam (Levene Test)
- ✅ Visualisasi boxplot per perlakuan dan per kelompok
- ✅ Visualisasi Q-Q Plot residual
- ✅ Uji ANOVA dua arah:
  - Pengujian faktor perlakuan
  - Pengujian faktor kelompok
- ✅ Keputusan dan kesimpulan statistik otomatis
- ✅ Uji lanjut post hoc:
  - BNT (Beda Nyata Terkecil / LSD)
  - BNJ (Beda Nyata Jujur / HSD Tukey)
  - Duncan
